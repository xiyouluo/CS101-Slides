# CS101-Slides

我的 2025 年秋季学期计算概论上机授课 Slides。

基于 [Slidev](https://sli.dev/) 构建，并使用 [Github Pages](https://pages.github.com/) 部署。

## 使用

```bash
git clone https://github.com/xiyouluo/CS101-Slides.git
cd CS101-Slides
pnpm install
pnpm run dev
```

你可能需要注释掉文件开头 `Formatter` 里的 `presenter: false` 以启用演讲者模式。
