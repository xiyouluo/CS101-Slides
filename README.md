# Arthals-ICS-Slides

我的 ICS 小班授课 Slides。

基于 [Slidev](https://sli.dev/) 构建。

## 使用

```bash
git clone https://github.com/zhuozhiyongde/Arthals-ICS-Slides.git
cd Arthals-ICS-Slides
pnpm install
pnpm run dev
```
